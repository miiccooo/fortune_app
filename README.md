# Fortune App

誕生日をもとに運勢を表示する占いWebアプリです。

---

## ■ 概要

星座・タロット・四柱推命（簡易）を組み合わせて、今日の運勢を表示します。
日常の中で気軽に楽しめる占い体験を意識して制作しました。

---

## ■ 使用技術

- Python（Flask）
- HTML / CSS

---

## ■ 主な機能

- 誕生日入力による星座判定
- タロットカードの表示（正位置・逆位置）
- 四柱推命（年柱）の算出
- ラッキーカラー表示
- 総合的なメッセージ生成

---

## ■ 工夫した点

- 複数の占いロジックを組み合わせて、独自の結果を出すように設計
- ユーザーの気持ちに寄り添うメッセージになるよう文章を調整
- 占いの世界観に合うようにデザインや構成を意識

---

## ■ 使い方

1. アプリを起動
2. 誕生日（YYYY-MM-DD）を入力
3. 今日の運勢が表示されます
